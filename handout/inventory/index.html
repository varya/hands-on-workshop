Download Figma file

Each team can choose one person to own the design file. This person can download Figma starter file here

Sharing the file

Once this person downloaded the file and opened it in Figma, this person can start sharing the file to his team members.

Access rights

  1. Click the Share button in the file.
  2. Click the to choose how people can find and access the file.
  3. Click the to set the permissions for sharing links.
  4. Check the box if you want the URL or embed code to Link to selected Frame (optional).
  5. Click Copy Link to copy the file URL to your clipboard. If you're using Figma in the browser, you can also copy the file link in the address bar.
