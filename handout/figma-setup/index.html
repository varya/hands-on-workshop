Figma Setup

A) Download Figma file

Each team can choose one person to own the design file. This person can download and open The Figma starter file here into her/his Figma account.

  1. Open Figma starter file
  2. Log in to Figma or create an account
  3. Click Figma logo dropdown
  4. Go to File/Save local copy
  5. Drag and drop the downloaded file to Figma

B) Sharing the file

The assigned person person can start sharing the file to his/her team members.

Access rights

  1. Click the Share button in the file.
  2. Click the to choose how people can find and access the file.
  3. Click the to set the permissions for sharing links.
  4. Check the box if you want the URL or embed code to Link to selected Frame (optional).
  5. Click Copy Link to copy the file URL to your clipboard. If you're using Figma in the browser, you can also copy the file link in the address bar.

  • Can edit: recommend for designers
