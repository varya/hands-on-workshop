## A) Download Figma file
Each team can choose one person to own the design file.
This person can download and open [The Figma starter file here](https://www.figma.com/file/GDxXj0FoQ2ppNCAMcPZpmV/Workshop-v2?node-id=113%3A3565)
  into her/his Figma account.  

  1. [Open Figma starter file](https://www.figma.com/file/GDxXj0FoQ2ppNCAMcPZpmV/Workshop-v2?node-id=113%3A3565)
  1. Log in to Figma or create an account
  1. Click Figma logo dropdown
  1. Go to **File/Save local copy**
  1. Drag and drop the downloaded file to Figma



![Figma starter file here](https://res.cloudinary.com/goright-io/image/upload/f_auto,q_auto/v1619277298/handout/figma-duplicate-file-v2_pdfftn.png)



### B) Sharing the file

The assigned person person can start sharing the file to his/her team members. 

#### Access rights

  1. Click the Share button in the file.
  1. Click the  to choose how people can find and access the file.
  1. Click the  to set the permissions for sharing links.
  1. Check the box if you want the URL or embed code to Link to selected Frame (optional).
  1. Click Copy Link to copy the file URL to your clipboard. If you're using Figma in the browser, you can also copy the file link in the address bar.


![Share figma access](https://help.figma.com/hc/article_attachments/360084499614/Link_sharing_settings_in_the_share_modal_for_a_file_in_a_team.png)


  - Can view: recommend for devs 
  - Can edit: recommend for designers 

### C) Plugins to install

  1. [Design Tokens by Lukas Opperman](https://www.figma.com/community/plugin/888356646278934516/Design-Tokens)
  1. [Themer by Tom Lowry](https://www.figma.com/community/plugin/731176732337510831/Themer)

#### Themer setup 
  1. [Create an account a https://jsonbin.io/](https://jsonbin.io/)
  1. Under profile avatar go to API Keys
  1. Copy Key
  1. Open themer panel inside your Figma
  1. Under tab admin add paste the API Key
